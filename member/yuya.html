Q&A

　Q : アルバに入った理由は？

　A : 小さい頃から鳥人間コンテストをずっと見てて、大学に入ったら鳥人間サークルに入りたいと思っていたから。飛行機作りたい！飛ばしたい！飛びたい！と思ったから。パイロットやりたいと思ったから

　Q : アルバでの思い出は？

　A : ・パイロットの代理で回転試験をしたとき・全組で飛行機の全貌を見たとき・TFのときの日の出・代表がキムワイプを大人買い(さすが代表！おれたちにできない事を平然とやってのける！そこにシビれる！憧れるゥ！)

　Q : 趣味・特技は？

　A : スキー、ピアノ、漫画(ヒロアカ、ワートリ、JOJO)、坂道系アイドル(乃木坂、欅坂、櫻坂、日向坂)、音楽鑑賞（特にUVERworld、MAN WITH A MISSION、[Alexandros]とか好きです）

　Q : 尊敬する人は？笑

　A : 田中代表ことキムワイプ代表(笑)

　Q : 私と言えば○○！

　A : 圧倒的鳥人間狂信者、ヘルメット

　Q : 私と言えば○○の理由

　A : ・先輩の紹介で入学式よりも前の3月中旬に入部した（この時点で相当鳥人間に取り憑かれてる、代表曰く｢恐ろしく早い入部、俺でも見逃しちゃったね｣）・過去に鳥人間コンテストに出場したチームの機体の特徴をある程度言える・夜にYoutubeで色んな鳥人間チームの過去のTFの記録を見続けてたら気づいたらオールしてたこともちょいちょいある・なんにも作業がないのに寝袋とスタイロ(翼の材料)の端材を使って部室に泊まるという普通ならしないであろうプレーを平然とやってのけた（意外と寝心地よかったです、なんなら家より過ごしやすかったまである）・組立試験とか桁の荷重試験で常にヘルメットを装着してる

　Q : これからアルバに入ってくれるかもしれない人へのメッセージをどうぞ！

　A : ｢飛行機を作りたい！飛ばしたい！飛びたい！ ｣の欲張り三段活用ができるのが鳥人間サークルの強みだと思います！｢うち文系やから理系の知識あらへんしキツそう…｣って思われがちですが、そんなことないです！｢工作が好き！｣｢飛行機が好き！｣という気持ちだけあれば十分です！困ったときは先輩やOB・OGさんが 0から教えてくれます！一緒に飛行機を作れることを楽しみにしてます！
