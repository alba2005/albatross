Q&A

　Q : アルバに入った理由は？

　A : 打算的には学チカ。後は、皆で喋りながらやる作業が楽しそうと思った

　Q : アルバでの思い出は？

　A : 12月のTFで凍え死にかけた。キムワイプを発注しすぎた。先輩の苦労（設計）を理解した。

　Q : 趣味・特技は？

　A : ゲーム・ボカロ・アニメ・散歩

　Q : 尊敬する人は？笑

　A : サカナクションの山口一郎、橘川さん、清水昭比古

　Q : 私と言えば○○！

　A : のっぽ。多分部の中では一番サブカルオタク。

　Q : 私と言えば○○の理由

　A : 背が高いから。もう一つの方は、ボカロオタ・アニオタ・ゲームオタの三拍子を俺以上に揃えてる奴を俺はこの部では知らない。

　Q : これからアルバに入ってくれるかもしれない人へのメッセージ をどうぞ！

　A : 最近キラキラめな雰囲気のある部ですが、俺みたいな奴だって結構いたりする。肩身の狭い俺に居場所を下さい。まぁ普通に賑やかで楽しいわよ。入れ。
